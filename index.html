Fusiongrid
FusionGrid is a highly intuitive and customizable JavaScript data grid component for both web and mobile applications. It allows you to build data grids in dashboards with the same simplicity and elegance as FusionCharts. This new release comes with exciting features like components integration, column filter support, support for data export, row selection, column grouping support, real-time data updates, row sorting, search filter support, and row animation upon data operation interaction.

Support for Integration Components, Data export.

With this release, we have introduced integration components for three primary front-end frameworks (React, Angular, and Vue) FusionGrid comes with a Data Export feature that enables you to get/download the data rendered on the grid. We have also introduced export functionality for CSV, JSON, and XLS (Excel) formats.

View Details
Support of filtering, Grouping/Nested Columns

FusionGrid comes with a column filter that enables you to easily filter the grid that will help you discover the data you are looking for based on the context. Now you can easily access the benefits of a column filter with both conditional and value filtering. We have incorporated Column Grouping in FusionGrid. It enables you to group the information on any of the relevant columns. This enhances the data experience and provides better data context.

View Examples
Support for Search/Quick Filter, Row Animation and more.

FusionGrid comes with a Search/Quick filter that enables you to search the information on any of the relevant columns so that you can filter the data quickly. This version is implemented with column search, external/API search, instant search, and pattern Search. With this release, we have introduced Row Animation. It enhances your experience when any data operation (sort, search, filter, etc.) interaction takes place as the transition effect.
